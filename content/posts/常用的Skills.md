## 通用

### find-skills

帮助发现和安装 agent skills。

```bash
npx -y https://jsr.io/@anthropic-ai/install-skill find
```

### nuxt-ui

使用 @nuxt/ui v4 构建 UI，提供 125+ 可访问的 Vue 组件和 Tailwind CSS 主题。

```bash
npx -y https://jsr.io/@anthropic-ai/install-skill install nuxt-ui
```

### slidev

使用 Markdown、Vue 组件、代码高亮、动画和交互功能创建和演示基于 Web 的幻灯片。

```bash
npx -y https://jsr.io/@anthropic-ai/install-skill install slidev
```

### large-task-executor

用于 Codex 的大任务执行器。

```bash
# 位于 ~/.codex/skills/large-task-executor
```
