react hooks 入门

编辑
2022-02-21
react • hooks

React在组件变得复杂之后使用Class组件会变得更难以维护, React Hook 是 React 16.8 的新增特性。它可以让你在不编写 class 的情况下使用 state 以及其他的 React 特性。通常来说hook使得在组件之间复用状态逻辑变得便捷、逻辑也更清晰。文章主要参考了React Hooks, 在此基础上添加了更多的用例。

API

基础hook

useState

在函数组件中存储内部 state

const [state, setState] = useState(initialState);

参数:

  • initialState: 初始值

返回值:

  • state: 当前的 state
  • setState: 更新 state 的方法, 它接收一个新的 state 值并将组件的一次重新渲染加入队列。setState(newState);, 如果你的更新函数返回值与当前 state 完全相同，则随后的重渲染会被完全跳过
